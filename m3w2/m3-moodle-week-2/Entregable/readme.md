# # Gestión de Datos en JavaScript
# 
# ## Descripción
# Este scrips usa objetos, sets y maps en JavaScript para manejar datos de productos (genéricos). 
# Está dividido en 5 secciones simples para manejar distintos escenarios o posibles usos:
# 1. Crear objeto de producto.
# 2. Usar Set para listas únicas.
# 3. Crear Map para categorías.
# 4. Iterar estructuras.
# 5. Validar y probar productos.
# 
# ## Requisitos
# - Navegador (como Chrome) con consola.
# - VS Code para ejecutar JS.
# 
# ## Cómo Ejecutar
# 1. Ejecuta el archivo index.html con un navegador basado en chromio
# 2. Abre el archivo index.html en un editor de código (visual studio code) y abre con live server
# 3. Usa la consola de tu editor de preferentica
# 
# Verás resultados en consola.
# 
# ## Estructura del Código
# - **Sección 1**: Objeto básico con id, nombre, precio.
# - **Sección 2**: Set para eliminar duplicados, con add, has, delete, for...of.
# - **Sección 3**: Map para categorías, función para agregar productos.
# - **Sección 4**: Iterar con for...in (objeto), for...of (Set), forEach (Map).
# - **Sección 5**: Validar productos (id string, nombre string, precio >0), filtrar lista, Set de nombres, Map de categorías.
